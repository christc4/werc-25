About

bq is a line-oriented text editor written in go, it draws inspiration from ed

bq is in alpha testing

Features

  • Fewest key-strokes
    • Simplemost operations require only one key-press
  • Configurable
    • Custom commands will* be defined in a header file, akin to suckless programs

Commands

  • [s] - saves file
  • [.] - saves and quits file
  • [p] - prints file
  • [!] followed by [system command] - uses shell to execute given command

Install

git clone https://git.bauherren.ovh/bq

go build -o bq *.go
